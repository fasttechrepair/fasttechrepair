# Fast Tech Repair Website

Professional GitHub Pages website for Fast Tech Repair.

## Files

- index.html
- styles.css
- script.js
- README.md

## GitHub Pages Setup

1. Create a new GitHub repository.
2. Upload all files from this folder.
3. Go to Settings.
4. Go to Pages.
5. Source: Deploy from a branch.
6. Branch: main.
7. Folder: /root.
8. Click Save.

Your website will be published by GitHub Pages.
